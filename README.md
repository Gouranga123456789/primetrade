# PrimeTrade Frontend Developer Task

A full-stack web app with authentication and a dashboard.

## Tech Stack
- React (Vite)
- TailwindCSS
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication + bcrypt

## Features
- User Register + Login (JWT)
- Protected Dashboard
- Profile Fetch + Update
- Tasks CRUD (Create, Read, Update, Delete)
- Search + Filter tasks
- Logout

---

## Setup Instructions

## Clone Repo
```bash
git clone https://github.com/Gouranga123456789/primetrade
cd PrimeTrade
```

## Setup Backend and run
```bash
cd server
npm install
npm run dev
```
## Install MongoDB Compass (GUI)
This helps you visually check:
    - users collection
    - tasks collection

## Setup Frontend and run
```bash
cd client
npm install
npm run dev
```
